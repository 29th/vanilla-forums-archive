    RS2 is built using the UE3 platform. I assume the HW requirements will fall somewhere in the range below.

    The minimum HW specs for UE3 (Users) are

    • Windows XP SP3 (32-bit only)
    • 2.0+ GHz processor
    • 2 GB system RAM
    • A graphics card with shader model 3.0 support.

    The Reqired HW specs for UE3 testers (internal) are

    • Custom built PC's with
      • Windows 7 64-bit
      • Intel Core2Extreme Quad Core Processor - Q6800 - 2.93 GHz
      • 4 GB DDR2 RAM
      • nVidia GeForce 8800 (768 MB GDDR3)
      • 1x320 GB Hard Drive (OS Drive), 1x500 GB Hard Drive (Data Drive)

    Recommended HW Specs. for UE3 Developers

    • Windows 7 64-bit
    • 2.0+ GHz multi-core processor
    • 8 GB System RAM
    • NVIDIA 8000 series or higher graphics card
    • Plenty of HDD space

    The large amounts of recoil is intentional. RO1 had that, along with significant weapon sway. The guy who made Dibblers mod for RO2 (aka Immersion Overhaul Mutator) was hired on and thus influenced that aspect of the game. Honestly I like it. Less point and shoot with instant kill and more patience, timing, etc. Also allows for some cool maneuvers that sometimes can't be done because of the "instagib"-ness of some games.

    Game feels like it has the better elements of DH and RO2 in terms of gameplay. No more "zoom" which is nice, you feel a bit stockier and thus slower like in DH, but its responsive like in RO2.
