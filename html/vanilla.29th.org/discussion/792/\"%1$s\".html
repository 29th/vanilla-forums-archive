Air to Air Combat Ready Checklist

  • Understands the 4 main AA missile types
  • Can fire heaters without radar
  • Can fire heaters with radar (DFGT MM) All aspect/rear aspect
  • Can understands the indexer and fire AMRAAMs
  • Understands the importance of beaming
  • Using TGP in AA mode to ID aircraft
  • Can perform a Split S and burn maneuver
  • Understands the fundamentals of breaking into missiles
  • Understands how to evade SAM missiles
  • Have basic Ground to Air threats knowledge
  • Can find targets using radar and lock them up
  • Can break through jamming equipment
  • Can fire multiple slammers at once using TWIS mode
  • Can turn on and off RWR
  • Can use HMCS to lock up targets off boresight
  • Can maintain comms discipline
  • Can set up a BINGO
  • Can use TCN
  • Understands the merry go round
  • Can make a flot and pre planned steer points

Air to Ground Combat Ready Checklist

  • Can use ground radar in GM and GMT
  • Can use snow plough ground radar
  • Can use TGP to locate targets
  • Can use mavericks as a stand off weapon
  • Can use CCIP and CCRP and lofting and pop up lofting
  • Can use TFR
  • Can use SEAD
  • Can use the HAD to bomb
  • Can use AGM88s without the HAD
  • Can use laser bombing
  • Can set up pre planned target steer points
